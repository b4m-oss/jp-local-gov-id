# Examples

Interactive examples in the browser (official dataset bundled). Each page includes a live demo and source code.

## Address input

Select a prefecture to load its municipalities into a dropdown. Styled like a typical address form.

[View the address input example](/en/examples/address-input)

## Municipality validation

With a prefecture selected, enter a municipality name. Validation runs when the field loses focus.

[View the municipality validation example](/en/examples/municipality-validation)

## Nationwide municipalities

Like a real-estate area filter: checkbox lists of every municipality, grouped by prefecture.

[View the nationwide municipalities example](/en/examples/nationwide-municipalities)
